## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Background.** bgdude records mood tags but never relates them to glucose (this resolves the open question in the mood-logging task).

**Reason for change.** A simple weekly correlation between mood and that day's time-in-range — shown only when it's statistically real, and explicitly showing "nothing significant yet" otherwise — turns mood logging from dead data into an occasional honest insight.
<!-- SECTION:DESCRIPTION:END -->

## Implementation Plan

<!-- SECTION:PLAN:BEGIN -->
- Compute a weekly Spearman correlation of mood tags vs same-day TIR/CV, requiring ≥8 tagged days.
- Show only significant results.
- If nothing significant after 8 weeks, explicitly show nothing ("nothing significant yet" empty state).
- Add/extend unit tests under `test/`: Spearman + significance unit test; "nothing significant" path renders the explicit empty state.
- Verify: `flutter analyze` clean, `flutter test` green.

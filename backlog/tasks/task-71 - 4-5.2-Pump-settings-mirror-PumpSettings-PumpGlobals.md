## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Background.** The pump stores its configuration — auto-shutdown timer, low-insulin warning level, screen timeout, cannula-fill amount, and quick-bolus setup — which bgdude can read but doesn't show. These read messages were confirmed to return real data during the July 2026 live-pump exploration (the Protocol Explorer sweep); their exact byte layouts are recorded in doc/pump-protocol.md.

**Reason for change.** Showing this as a read-only "pump configuration" card lets you verify your settings from the app and enables reminders (for example, an auto-shutdown countdown).

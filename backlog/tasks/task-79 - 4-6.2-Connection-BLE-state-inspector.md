## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Background.** When pump pairing or reconnection misbehaves, the only way to see what's happening today is to plug the phone into a computer and read low-level logs.

**Reason for change.** An on-device connection inspector — showing the connection stage, bond state, pairing progress, firmware, reconnect counts and last error — makes those problems legible on the phone itself, which is invaluable for the pump-reliability and software-pump work.
<!-- SECTION:DESCRIPTION:END -->

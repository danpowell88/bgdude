## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Background.** The forecast is a black box — you can see the predicted line but not what's driving it (insulin? carbs? momentum?).

**Reason for change.** An advanced-mode breakdown that attributes the forecast to insulin vs carbs vs momentum, plus the model's correction and where the band comes from, builds trust and helps debugging. It's cheap — just a few quick "what if this input were zero" runs.

## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Background.** The pump reports which features are actually enabled on it (which CGM, whether Control-IQ is on, whether Bluetooth control is allowed) and its localization (units/region). bgdude currently assumes rather than checks. These read messages were confirmed to return real data during the July 2026 live-pump exploration (the Protocol Explorer sweep); their exact byte layouts are recorded in doc/pump-protocol.md.

**Reason for change.** Reading the real feature set lets bgdude hide or show parts of the UI to match what your specific pump supports, instead of guessing.
<!-- SECTION:DESCRIPTION:END -->

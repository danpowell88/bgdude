## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Background.** The pump can stream its history log (boluses, basal changes, alarms, site changes, CGM readings — 130+ event types). bgdude decodes only some of them, best-effort.

**Reason for change.** A raw history-log viewer with a coverage report (which event types decode vs fall through) helps expand and verify the history decoding against a real pump.

## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Background.** bgdude now has a hidden "Developer" menu for low-level, read-only diagnostics. This adds a live feed of every message and event coming from the pump (name, code, raw bytes, decoded values), separate from the Protocol Explorer's send-a-request-and-see flow.

**Reason for change.** A running message monitor makes it easy to watch what the pump is actually sending during normal operation — useful for debugging decode issues and understanding the protocol.
<!-- SECTION:DESCRIPTION:END -->

<!-- SECTION:PLAN:BEGIN -->

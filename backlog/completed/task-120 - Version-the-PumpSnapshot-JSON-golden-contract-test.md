## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**Background.** The snapshot wire format is hand-rolled on both sides (`android/.../pump/PumpTypes.kt:71-106` `MutableSnapshot.toJson`, `lib/pump/pump_snapshot.dart:160-179` `fromJson`) with no version field; adding/renaming/retyping a field silently yields null on the Dart side.

**Reason for change.** The Kotlin-Dart contract is the main cross-language seam; a schema version plus a golden fixture turns silent drift into a failing test.
<!-- SECTION:DESCRIPTION:END -->
